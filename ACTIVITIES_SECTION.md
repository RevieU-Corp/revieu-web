### 🎨 ActivityCard 组件设计

#### 视觉特点:
- **尺寸**: 256×128px (16:8 比例)
- **圆角**: `rounded-2xl` 现代化外观
- **阴影**: 悬停时增强阴影效果
- **背景**: 高质量美食图片作为背景
- **遮罩**: 黑色半透明遮罩确保文字可读性

#### 交互效果:
- **悬停缩放**: `hover:scale-105` 轻微放大
- **点击反馈**: `active:scale-95` 点击时缩小
- **阴影变化**: 悬停时阴影加深
- **光泽效果**: 内置光泽动画 (暂时隐藏)

#### 内容布局:
```
┌─────────────────────────────────┐
│ [BADGE]                         │ ← 左上角徽章
│                                 │
│                                 │ ← 背景图片
│                                 │
│ Title                           │ ← 底部标题
│ Subtitle                        │ ← 副标题
└─────────────────────────────────┘
```

### 📱 响应式滚动设计

#### 滚动特性:
- **横向滚动**: `overflow-x-auto` 支持左右滑动
- **隐藏滚动条**: `no-scrollbar` 保持界面简洁
- **卡片间距**: `gap-4` 适当的间距
- **边距处理**: `-mx-3 px-3` 滚动区域延伸到边缘

#### 移动端优化:
- **触摸友好**: 卡片大小适合手指滑动
- **流畅滚动**: 原生滚动体验
- **边界处理**: 左右边距确保内容不贴边

### 🏷️ 徽章系统

#### 徽章类型和颜色:
| 类型 | 颜色 | 含义 |
|------|------|------|
| WEEKLY | 橙色 (`bg-orange-500`) | 每周活动 |
| LIMITED | 蓝色 (`bg-blue-600`) | 限时优惠 |
| STUDENT | 红色 (`bg-red-600`) | 学生专享 |
| DAILY | 绿色 (`bg-green-500`) | 每日活动 |
| WEEKEND | 紫色 (`bg-purple-500`) | 周末特惠 |

#### 徽章设计:
- **位置**: 左上角固定位置
- **样式**: 圆角矩形，白色文字
- **字体**: 粗体小号字体
- **阴影**: 无额外阴影，依赖背景对比

### 🖼️ 图片选择策略

#### 图片来源:
- **Unsplash**: 高质量免费图片
- **主题匹配**: 每张图片都与活动内容相关
- **色彩丰富**: 选择色彩鲜艳的图片增加吸引力

#### 图片处理:
- **背景覆盖**: `bg-cover bg-center` 确保图片填满容器
- **遮罩层**: `bg-black/30` 30%透明度黑色遮罩
- **文字阴影**: `textShadow` 确保文字在任何背景上都清晰可读
